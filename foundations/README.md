# Foundations

Foundations are rigorous grounding paths for mathematical statements.

Riemann Atlas should eventually support several foundations side by side, including:

- computational foundations based on natural numbers, rational numbers, algorithms, approximants, and limits
- classical real analysis and set-theoretic foundations
- algebraic abstraction paths such as real numbers to fields to rings to groups
- proof-assistant foundations such as Lean/mathlib encodings

A foundation explains how a statement is made precise. It should not automatically become the ontology of the whole atlas.
